Detailed Description

Defines shortcut with native keycodes.

The NativeShortcut holds the native keycode and modifiers, after they have been translated from Qt::Key and Qt::Modifiers into the native ones.

It can be used to find out how a current hotkey is mapped to the system, or to explicitly create a shortcut from those native values
